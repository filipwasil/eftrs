# EFTrs

Escape From Tarkov helper app written in Rust.

Necessary packages:
```
cargo
rustup
```

Build:
```
  cargo build
```

Usage:
```
  cargo run ADAR215
```
Output:
```
  WEAPON: ADAR215
  BUDGET: "5.56x45 mm M856A1"
  THE BEST: "5.56x45 mm M995"
  SPECIAL: "5.56x45 mm M995"
```


Current features:
- suggesting an optimal rounds for budget, best, and special loudouts
